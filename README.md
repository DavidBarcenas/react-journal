# Journal App

It gives you a quick and easy editing notepad experience when writing notes.

## Preview
![](/.readme-static/app.jpg)


## Features

* Login with google
* Redux to handle state
* Firebase

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run start
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```
